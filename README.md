# Customer Analytics Dashboard (Using sql and Tableau)

## Project Overview

Analyzed customer, order, and billing data to uncover insights on customer behavior, retention, and revenue performance.

## Objectives

- Analyze revenue trends and product performance

- Segment customers using RFM analysis

- Measure retention using cohort analysis
  
- Identify churn risk and high-value customers

## Tools Used

- SQL (MySQL)

- Tableau
  
- Excel (data cleaning)

## Dataset

- Contacts table (customer info)
  
- Orders table (transactions)
  
- Billing table (payments & refunds)

## Key Analyses

### 1. Revenue & Sales Trends

- Total revenue and order growth over time

- Average order value (AOV)
  
### 2. Cohort Analysis
   
- Customer retention by acquisition month
  
- Identified drop-off trends after first purchase
  
### 3. RFM Segmentation
   
Segmented customers into:

- Champions
  
- Loyal Customers
  
- At Risk
  
- Hibernating
  
### 4. Churn Analysis
Built churn risk model based on:

1. Recency
   
2. Frequency

3. Refund behavior

- Jean Ribault High School products drive majority of revenue

## Business Recommendations

- Target “At Risk” customers with retention campaigns
  
- Continue to invest in high-performing acquisition channels
